# Puzzle-15-
Пятнашки — головоломка с 15 пронумерованными костяшками в квадратной сетке 4×4, где одно поле пустое. Цель — упорядочить числа по возрастанию (от 1 до 15), перемещая костяшки на свободное поле. Каждый ход — сдвиг соседней костяшки в пустую клетку. Задача — решить за минимальное число ходов. Игра развивает логику и пространственное мышление.
